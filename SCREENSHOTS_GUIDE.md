**Cíl:** Vytvořit profesionální screenshoty pro iOS App Store a Google Play Store  
**Verze:** 3.0.0

## 📱 Technické Požadavky

### iOS App Store
- **Rozlišení:** 1242 × 2208 px (iPhone 6.1")
- **Počet:** 5-8 screenshotů (povinně alespoň 1)
- **Formát:** PNG nebo JPG
- **Obsah:** Čeština + English (pro different locales)

### Google Play Store
- **Rozlišení:** 1080 × 1920 px (Pixel 5.5")
- **Počet:** 4-8 screenshotů
- **Formát:** PNG nebo JPG
- **Obsah:** Čeština + English

## 🎯 Jak Nafotit / Vytvořit Screenshoty

### Metoda 1: Přímo z Telefonu (Nejjednodušší)

#### iOS:
```bash
# 1. Spusť app v Xcode na simulátoru (nearest to 6.1")
# 2. Simulátor: Device → Set Status Bar... → Edit...
# 3. Vytvoř nějaké data (spusť měření, sesbírrej výsledky)
# 4. Cmd + S = Screenshot (automaticky na Desktop)
# 5. Obrázky: ~/Desktop/
```

#### Android:
```bash
# 1. flutter run na emulátoru (Pixel 5.5")
# 2. Emulator: Open Extended Controls → Screenshot
# 3. Nebo: adb shell screencap -p /sdcard/screen.png
#    adb pull /sdcard/screen.png ~/Desktop/
```

### Metoda 2: Android Studio / Xcode Built-in

**iOS (Xcode):**
1. Spusť simulator
2. Simulator menu → File → New Screenshot
3. Automaticky se uloží na desktop

**Android (Android Studio):**
1. Run app na emulátoru
2. Logcat → Devices File Explorer
3. /data/local/tmp/screenshots/ (pokud app má permission)
4. Nebo: Device Manager → Screenshot button

### Metoda 3: Programovací Approach (Integration Tests)

```bash
# Flutter integration_test framework
cd tepovka
flutter test integration_test/app_test.dart --dart-define USE_SCREENSHOT=true

# Screenshots se uloží do: build/screenshots/
```

## 🖼️ Design Tips

### Vstupování Pro Oba Platformy:

| Aspekt | iOS | Android |
|--------|-----|---------|
| **Font** | SF Pro Display | Roboto |
| **Barvy** | Sytější | Trochu templáty |
| **Okraje** | Zaoblené (notch) | Pravoúhlé |
| **Text Size** | Řekneš si sám | Řekneš si sám |
| **Button** | Zaoblené | Material Design |

### Návrh:
- **Text:** Jasný, velký, bez drobného textu
- **Barvy:** Loga blue/red pro srdeční pulz
- **Contrast:** Vysoký (musí se číst na malém displeji)
- **Jazyk:** Čeština pro CZ lokalizaci, English pro globální

## 📋 Checklist k kontrole

- [ ] 5-8 screenshotů per platforma
- [ ] Správné rozlišení (1242×2208 iOS, 1080×1920 Android)
- [ ] Čeština + English verze
- [ ] PNG/JPG kvalita ≥ 90%
- [ ] Žádné osobní údaje na screenshotech
- [ ] Nejnižší 2-3 screenshoty mají největší dopad → nejvíc péče
- [ ] Bez watermarků / date/time stamps
- [ ] Všechny viditelné prvky jsou jasné
